# Vocabulary data sources & attribution

`vocab.json` is generated by [`scripts/build_vocab.py`](../../../scripts/build_vocab.py)
from the following sources. Each card is a factual word→translation pair with a
CEFR level; where possible the article, example sentence, and English gloss come
from these sources.

| Source | Used for | License |
| --- | --- | --- |
| [Goethe-Institut A1/A2/B1 wordlists](https://github.com/ilkermeliksitki/goethe-institute-wordlist) (from the official Goethe Wortlisten) | Word selection, CEFR level, articles, example sentences | Official Goethe exam vocabulary |
| [FreeDict `deu-eng`](https://freedict.org/) | English glosses (sense-disambiguated) | **GPL-2.0** |
| [OpenSubtitles `de_50k`](https://github.com/hermitdave/FrequencyWords) | Frequency ordering | CC-BY-SA 4.0 |

The A1 glosses and ~200 common words were hand-corrected in the generator's
`OVERRIDES`/`EXTRA_A1` tables. B2/C1 breadth is drawn from the previous corpus's
lemmas, re-glossed via FreeDict.

**Note on licensing:** FreeDict `deu-eng` is distributed under the GPL-2.0.
Bare word-equivalence glosses are largely factual, but if you prefer to avoid
GPL provenance entirely, regenerate `vocab.json` with an alternative gloss
source (e.g. a CC-BY dictionary or hand-authored translations) — the generator
is structured so only the gloss step needs swapping.
