# Phase 2 Learning Materials

This directory contains Phase 2 learning materials for German language learning.

## Phase 2 Focus Areas

Phase 2 continues from Phase 1 and focuses on:
- Daily routines (Alltagsroutine)
- Separable, inseparable, and reflexive verbs
- Time expressions (12h / 24h clock)
- Reading comprehension (Herr Ihßen – Ein Tag im Leben eines Journalisten)
- Retrieval practice and active recall exercises
- Short writing and correction tasks

## Materials

The following materials are referenced in Phase 2:

1. **sokrates hausaufgabe.docx** - Homework assignments related to daily routines
2. **Retrieval Practice – Tägliche Routine.pdf** - Practice exercises for daily routines
3. **Herr Ihßen.pdf** - Reading comprehension text about a journalist's day
4. **Herr Ihßen (1).pdf** - Additional reading comprehension materials
5. **Meine Alltagsroutine.pptx** - Presentation on daily routines
6. **Test Your German.pdf** - Assessment materials

## Content Progression

Phase 2 follows a clear progression:
1. Basic routine verbs (aufstehen, sich waschen, sich anziehen)
2. Applied reading with context (Herr Ihßen story)
3. Production tasks (writing and speaking exercises)

## Note

These materials build upon Phase 1 fundamentals and introduce more complex verb structures and real-world reading contexts.
